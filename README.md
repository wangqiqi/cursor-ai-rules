## 🎯 快速开始

### 1. 克隆规则包
```bash
git clone https://github.com/wangqiqi/cursor-ai-rules.git
cd cursor-ai-rules
```

### 2. 部署到你的项目
```bash
# 复制到你的项目根目录
cp -r .cursor ~/your-project/

# 进入项目目录
cd ~/your-project

# 运行自动适配脚本
./.cursor/cursor-adaptation-setup.sh
```

### 3. 开始使用
规则系统自动生效！在Cursor中享受智能的AI协作体验。
